PercentNegativePattern

Gets or sets the format pattern for negative percent values.

OutputVar := instance.PercentNegativePattern
instance.PercentNegativePattern := Value

Value

MfInteger instance or var containing integer.

Gets

Gets the format pattern for negative percent values as an integer var.
The default for InvariantInfo is 0 which represents "-n %", where "%" is the PercentSymbol and n is a number.

Sets

Sets the format pattern for negative percent values.

Throws

Throws MfArgumentOutOfRangeException if the property is being set to a value that is less than 0 or greater than 11.
Throws MfInvalidOperationException if the property is being set and the MfNumberFormatInfo is read-only.

Remarks

The PercentNegativePattern property is used with the "P" standard format string to define the pattern of negative percentage values. For more information, see Standard Numeric Format Strings. This property has one of the values in the following table. The symbol "%" is the PercentSymbol, the symbol "-" is the NegativeSign, and n is a number. Values 3-11 were introduced in the .NET Framework 2.0 and cannot be used in earlier versions.

Value

1

2

3

4

6
