MfIndexOutOfRangeException Class

(Inherits from MfSystemException)

The exception that is thrown when an attempt is made to access an element of an array with an index that is outside the bounds of the array

Links

Properties, Methods, Constructor()

Example

GetIndexItem(index) {
   if (index > this.m_count) {
       err := new MfIndexOutOfRangeException("index out of range")
       err.SetProp(A_LineFile, A_LineFile, A_ThisFunc)
       throw err
   }
   return this.m_List[index]
}
