MfInvalidCastException Class

(Inherits from MfSystemException)

The exception that is thrown for invalid casting or explicit conversion.

Links

Properties, Methods, Constructor()

Example

MyFunction(objString) {
   ; expecting a instance of MfString class so lets check
   if (!MfObject.IsObjInstance(objString, MfString)) {
       Ex := new MfInvalidCastException("Unable to cast argument to MfString instance")
       Ex.SetProp(A_LineFile, A_LineFile, A_ThisFunc)
       throw Ex
   }
   ; do other processing
}
