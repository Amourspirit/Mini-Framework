GetValues()

Gets a MfGenericList of MfInteger containing values of derived MfEnum class.

Returns

Returns a MfGenericList of MfInteger containing values of derived MfEnum class.

Remarks

Each MfInteger instance in MfGenericList has it MfInteger.ReturnAsObject value set to false.

Example

; create new Enum Instance
myEnum := new RegOptEnum()

myValues := myEnum.GetValues()
; myValues is now a Generic collection of MfInteger

; myValues.Count will contain 18
MsgBox % myValues.Count

str := new MfString()

for i, IntObj in myValues
{
   str.Append(IntObj.Value)
   if ( i < (myValues.Count - 1)) {
       str.Append(",")
   }
}

;str.Value will contain 6,10,8,14,12,7,11,9,15,13,5,1,3,17,4,0,2,16
MsgBox % str.Value
