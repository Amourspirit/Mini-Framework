MfStack Class

(Inherits from MfEnumerableBase)

Represents a simple last-in-first-out (LIFO)  of objects.

Links

Properties, Methods

Example

mfSk := new MfStack()
i := 0
iEnd := 5
While, (i < iEnd)
{
   mss := new MfString()
   mss.Value := MfString.Format("String Value: '{0}'", i)
   mfSk.Push(mss)
   i++
}
mfsOutput := new MfString()
While mfSk.Count > 0
{
   mfsOutput.AppendLine(mfSk.Pop())
}
MsgBox, 64, Text, % mfsOutput.Value
/*
String Value: '4'
String Value: '3'
String Value: '2'
String Value: '1'
String Value: '0'

*/
