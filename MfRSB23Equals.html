Equals()

Inherits from MfObject.

OutputVar := instance.Equals(obj)

Equals(obj)

Compares obj to current instance to see if they are the same

Parameters

obj

The object to compare with current instance

Returns

Returns var containing Boolean value of true if the objects are the same; Otherwise false.

Remarks

If Objects have the same memory address then they are considered to be equal.

If you want to be certain that objects are of the same memory address use MfObject.ReferenceEquals()
