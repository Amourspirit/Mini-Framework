MfSystemException Class

(Inherits from MfException)

Defines the base class for predefined exceptions

Links

Properties, Methods, Constructor()

Example

Myfunction(x) {
   try
   {
       DoSomething(x)
   } catch e {
       err := new MfSystemException("Error, DoSomething did not succeed", e)
       err.SetProp(A_LineFile, A_LineNumber, A_ThisFunc)
       throw err
   }
}
