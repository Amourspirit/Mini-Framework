GetBaseException()

GetBaseException() Gets the MfException or derived object that is the root cause of one or more subsequent exceptions.
Inherited from MfException

instance.GetBaseException()

Returns

The first exception thrown in a chain of exceptions. If the InnerException property of the current exception is a null reference this property returns the current exception.

Throws

MfNullReferenceException if instance is null

Remarks

When overridden in a derived class, returns the MfException that is the root cause of one or more subsequent exceptions.
