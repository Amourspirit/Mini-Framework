MfNibbleList Class

(Inherits from MfListBase)

MfNibbleList is a sealed class that exposes methods and properties used for list of nibble values (0 - 15).

MfNibbleList is structured to work with nibbles ( values between 0 and 15 ) only. For a list that is better suited for containing objects see MfList and MfGenericList.
