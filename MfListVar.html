MfListVar Class

(Inherits from MfListBase)

MfListVar class exposes methods and properties used for common List and array type operations for collections.

MfListVar is structured to work with vars only and does no object checking in any of it methods. For a list that is better suited for containing objects see MfList and MfGenericList.
