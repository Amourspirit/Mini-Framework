Item

Overrides MfListBase.Item
Gets or sets the element at the specified index.

OutputVar := instance.Item[index]
instance.Item[index] := Value

Parameters

index

The zero-based index of the element to get or set.

value

the value of the item at the specified index

Gets

Gets element at the specified index.

Sets

Sets the element at the specified index

Throws

Throws MfArgumentOutOfRangeException if index is less than zero or index is equal to or greater than Count
Throws MfArgumentException if index is not a valid MfInteger instance or valid var containing Integer

Example

; MfList has a zero based index
myList := new MfList()
myList.Add(new MfChar("T"))
myList.Add(new MfString("Item 2"))
myList.Add(new MfString("Item 3"))
myList.Add(new MfBool(True))
myList.Add(7)

int := new MfInteger(1)
myList.Item[int] := new MfString("Hello World")
myList.Item[2] := new MfChar("V")
Msgbox % myList.Item[0].Value ; displays "T"
Msgbox % myList.Item[int].ToString() ; displays "Hello World"
Msgbox % myList.Item[2].Value ; displays "V"
Msgbox % myList.Item[3].ToString() ; displays "True"
Msgbox % myList.Item[4] ; displays "7"
