TryParse()

OutputVar := MfByte.TryParse(byte, obj)

TryParse(byref byte, obj)

Converts the obj representation of a number to its Integer equivalent. A return value indicates whether the conversion succeeded.

Parameters

byte

Contains the value equivalent to the number contained in obj, if the conversion succeeded.
The conversion fails if the obj parameter is not of the correct format, or represents a number less than MinValue or greater than MaxValue.

obj

An object convert.
Can be instance of MfChar, MfString
Can be var containing string, integer, hex value
Obj can also be instance of MfParams.

Returns

Returns true if parse is a success; Otherwise false

Throws

Throws MfInvalidOperationException if not called as static method.

Remarks

If parameter int is passed in as an object it must in initialized as an instance of MfByte first. Eg: myByte := new MfByte(0)
If parameter int is passed in as var it must in initialized as integer first. Eg: myByte := 0
White spaces are allowed at the beginning or end of the string to parse.

When parsing a MfChar instance the MfChar instance must be Numeric to parse successfully.
See MfChar.IsNumber.

Example

; this example uses objects but
; fails to parse as -1056 is outside the range of MfByte
pStr := new MfString("-1056")

; must be initialized to instance of MfByte before call to TryParse
myInt := new MfByte(0)
if (MfByte.TryParse(myInt, pStr)) {
    val := myInt.Value
    MsgBox The value is in pStr is:%val%
} else {
    MsgBox Unable to parse any value from pStr
}

; this example uses objects
; this example to will parse to a value of 88
pStr := new MfString("88  ")

; must be initialized to instance of MfByte before call to TryParse
myInt := new MfByte(0)
if (MfByte.TryParse(myInt, pStr)) {
    val := myInt.Value
    MsgBox The value is in pStr is:%val%
} else {
    MsgBox Unable to parse any value from pStr
}

; this example does not use objects
; this example to will parse to a value of 51
pStr := " 0x33"

; must be initialized to integer before call to TryParse
myInt := 0
if (MfByte.TryParse(myInt, pStr)) {
    MsgBox The value is in pStr is:%myInt%
} else {
    MsgBox Unable to parse any value from pStr
}

; this example parses a MfChar instance
; this example to will parse to a value of 9
mfc := new MfChar("9")
; must be initialized to integer before call to TryParse
myInt := 0
if (MfByte.TryParse(myInt, mfc)) {
    MsgBox The value is in pStr is:%myInt%
} else {
    MsgBox Unable to parse any value from pStr
}
