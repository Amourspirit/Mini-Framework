Item

Gets or sets the element at the specified index.
Inherited from MfCollectionBase.

OutputVar := instance.Item[index]
instance.Item[index] := Value

Parameters

index

The zero-based index of the element to get or set.

value

the value of the item at the specified index

Throws

Throws MfArgumentOutOfRangeException if index is less than zero.-or index is equal to or greater than MfParams.Count
Throws MfArgumentException if index is not a valid Ineger object or valid var containing MfInteger

Gets

Gets element at the specified index.

Sets

Sets the value at the specified index

See Also

See Also:MfString
See Also:MfInteger
See Also:MfChar

Example

int := new MfInteger(3)
myList.index[int] = new MfString("Hello World")
myList.Index[4] = new MfChar("V")
Msgbox % myList.Index[int].ToString() ; displays Hello World
Msgbox % myList.Index[4].Value ; displays V
