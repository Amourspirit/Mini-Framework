ToString()

Gets a string representation of the object.

Returns

Returns string value representing current instance values in the collection.

Throws

Throws MfNullReferenceException if called as a static method.

Example

mfCol := new MfCollection()
i := 0
iEnd := 10
While, (i < iEnd)
{
   if (Mod(i, 2)) ; only odd numbers
   {
       mss := new MfString()
       mss.Value := MfString.Format("I am odd number: '{0}'", i)
       mfCol.Add(mss)
   }
   else
   {
       msi := new MfInteger(i)
       mfCol.Add(msi)
   }
   i++
}

cStr := mfCol.ToString()
MsgBox %cStr%

; cStr contains
/*
0: {MfInteger} 0
1: {MfString} I am odd number: '1'
2: {MfInteger} 2
3: {MfString} I am odd number: '3'
4: {MfInteger} 4
5: {MfString} I am odd number: '5'
6: {MfInteger} 6
7: {MfString} I am odd number: '7'
8: {MfInteger} 8
9: {MfString} I am odd number: '9'
*/
