Equals(x, y)

Determines whether the specified objects are equal.

Abstract Method

Parameters

x

The first MfObject to Compare.

y

The second MfObject to compare.

Returns

Returns true if the specified objects are equal; otherwise, false.

Throws

Throws MfNotImplementedException if not overridden in derived class.

Remarks

Implement this method to provide a customized equality comparison for objects.

Notes to Implementers
The Equals method is reflexive, symmetric, and transitive. That is, it returns true if used to compare an MfObject with itself; true for two MfObject x and y if it is true for y and x ; and true for two MfObject x and z if it is true for x and y and also true for y and z.

Implementations are required to ensure that if the Equals method returns true for two objects x and y, then the value returned by the GetHashCode method for x must equal the value returned for y.
