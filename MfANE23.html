MfArgumentNullException Class

(Inherits from MfArgumentException)

The exception that is thrown when a null reference is passed to a method that does not accept it as a valid argument.

Links

Properties, Methods, Constructor()

Example

str := MfString("Hello World") ; str will be null as we did not use a new constructor
Myfunction(x) {
   if (!IsObject(x))
   {
       err := new MfArgumentNullException("x","Argument it null",)
       err.Source := A_ThisFunc
       throw err
   }
   try {
       retval := DoSomething(x)
   } catch e {
       err := new MfException("Error, DoSomething did not succeed", e)
       err.SetProp(A_LineFile, A_LineNumber, A_ThisFunc)
       throw err
   }
   return retval
}
